# Promotion Gate Checklist

Use this checklist before any item is accepted into `codex\outputs\standard-harness\`.

## Required Checks

- Is the item necessary to the harness philosophy?
- Is the item non-redundant relative to accepted outputs?
- Has project-specific correction been completed?
- Has missing detail been supplemented where needed?
- Has wording been normalized for this project?
- Have conflicts with accepted outputs been resolved?
- Is the final item clean enough to be treated as canonical?
- Is the maintenance cost justified by the value?

## Gate Rule

If any answer is no, do not promote the item.

Keep it in `codex\outputs\harness-candidates\` until the missing work is complete.
