# Standard Harness Outputs

This directory stores standard harness deliverables produced by this project.

These outputs are durable and reusable. They should be organized by purpose rather than by date.

Only accepted outputs belong here.

If an item still needs correction, supplementation, or conflict resolution, keep it in `..\harness-candidates\` until that work is complete.

## Categories

- `design\`: durable design outputs
- `templates\`: reusable templates and scaffolds
- `playbooks\`: reusable operational procedures
- `checks\`: reusable review and verification checklists
